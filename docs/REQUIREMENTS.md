# 우아한테크코스 - level1 - 자동차 경주

### 입출력 구현 목록

- [ ] 경주할 자동차 이름 입력 받기 
  - [ ] 이름은 , 로 구분
  - [ ] 이름 길이는 1 ~ 5자 이하만 가능, 아닐 시 에러 출력 후 재입력 
  - [ ] 중복 이름 입력 시 에러 출력 후 재입력
- [ ] 시도할 횟수 입력 받기 
  - [ ] 1회 이상 가능, 아닐 시 에러 출력 후 재입력
  - [ ] 정수 아닐 시 에러 출력 후 재입력
- [ ] 실행 결과 횟수만큼 출력
- [ ] 최종 우승자 출력 
  - [ ] 2명 이상일 시 , 로 구분 

### 기능 구현 목록

- [ ] 횟수만큼 모든 자동차에 대한 전진 여부 확인 
- [ ] 자동차 당 랜덤 숫자 (0 ~ 9) 발생하여 4 이상일 시 전진 
- [ ] 자동차의 현재 위치 저장
- [ ] 최종 우승자 판단
