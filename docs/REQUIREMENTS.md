# 기능 목록

1. 이름을 입력한다.
   - 이름 쉼표 기준 구분, 5자 이하
2. 횟수를 입력한다.
   - 회수 숫자가 아니면 에러 출력
3. 이동 / 정지 판단
   - 0 ~ 9 사이 무작위 값, 4 이상 이동
4. 이동 회수 출력

5. 판단, 출력 과정을 입력한 회수만큼 반복

6. 우승자 판별
