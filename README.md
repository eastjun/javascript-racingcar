# 기능목록

- 입력 : `경주할 자동차 이름을 입력하세요(이름은 쉼표(,)를 기준으로 구분).`
  - 쉼표로 구분하여 배열에 저장
  - 유효성 검사
    - 5자 이하
    - 쉼표를 기준으로 구분
  - (예외) 5자 초과
  - (예외) 0자
  - 실패했을경우 돌아감
- 입력 : `시도할 횟수는 몇 회인가요?`

  - 유효성 검사
    - 양의 정수
  - (예외) 양의 정수가 아닌 값
  - 실패했을경우 돌아감

- 출력 : `실행 결과`
- (반복)현재 게임 상황 출력
  - 각차마다 0부터 9사이의 무작위 값을 구함.
  - 그 값이 4이상이라면 전진한다. 아니라면 정지한다.
  ```
  pobi : -
  woni :
  jun : -
  ```
- 최종우승자 출력
  - 우승자를 쉼표로 나누어 출력(여러명일 경우)
  ```
  최종 우승자: pobi, jun
  ```
