# javascript-racingcar

### 1. 자동차 이름 입력 ✅

-✅ 유효성 검사

- (공백 제거 후) 자동차 이름의 길이가 5자 이하여야 한다.
- 자동차들은 쉼표로 구분되어야 한다.
- 자동차는 두 대 이상이어야 한다.
- 자동차 이름은 중복되지 않아야 한다.
- 빈 값이 포함되지 않아야 한다.

- ✅ 자동차 전진 기능

### 2. 시도 횟수 입력 ✅

- ✅ 유효성 검사
  - 숫자여야 한다.
  - 0 초과여야 한다.
  - 정수여야 한다.
  - (optional) 숫자가 너무 크지 않아야 한다.

### 3. 실행 결과 출력

- ✅ 자동차 경주

  - 무작위 값 받아오기
  - 자동차 전진 함수 호출

- 경주 결과 출력
  - 매 경주마다 자동차 위치 출력
  - 모든 경주를 마친 후 자동차 위치 기록을 출력

### 4. 최종 우승자 출력

- 최종 우승자 계산

- 최종 우승자 출력
