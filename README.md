# javascript-racingcar

## 기능 구현 목록
1. 자동차 이름을 입력받은 후 자동차 생성
- 쉼표를 기준으로 자동차 이름을 입력받는다.
- 입력받은 자동차 이름을 기반으로 자동차 객체 생성
#### 예외 사항
   - 이름 길이 5글자 초과
   - 공백 입력
2. 시도할 횟수를 입력 받는다.
   #### 예외 사항
   - 공백 입력
   - 0 이하의 숫자
   - 숫자가 아닌 입력
   - 정수가 아닌 소수 입력
3. 각 자동차의 전진 조건 구하기
   - 0에서 9 사이에서 무작위값이 4 이상일 경우, 전진
   - 3 이하인 경우 멈춤
4. 회차별 실행 결과 출력
5. 시도할 횟수만큼 3번, 4번 기능 반복
6. 가장 멀리간 자동차 구하기
   - 한명 이상 가능
7. 최종 우승자 출력
