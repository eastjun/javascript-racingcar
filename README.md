# javascript-racingcar

# 기능 목록

1. 자동차 이름 입력

- [x] 자동차 이름 입력.
- [x] 쉼표(,)를 기준으로 구분
- [x] 1~5자가 아닌 것에 대한 예외처리
- [x] 이름 중복 예외처리
- [x] 빈 입력에 대한 예외처리
- [x] 자동차 수가 한명일 경우 예외처리

2. 시도 횟수 입력

- [x] 시도횟수 입력.
- [x] 시도횟수를 숫자로 변환
- [x] 20 이하가 아닌 것에 대한 예외처리
- [ ] 자연수 입력이 아닌 것에 대한 예외처리

3. 전진하는 조건 설정

- [ ] 각 게임별 랜덤 숫자 설정 (0~9 설정)
- [ ] 4 이상 숫자에서 전진
- [ ] 시도횟수 만큼 반복

4. 실행 결과 출력

- [ ] 각 시도횟수별 실행 결과 출력

5. 우승자 선별

- [ ] 우승자 선별
- [ ] 다수의 우승자일 경우, 쉼표로 구분해서 출력. 단일 우승자의 경우, 이름만 출력
